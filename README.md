# Sobre este curso

## :dvd: Curso de SQL: Business Intelligence, SQL Server, MySQL, Oracle, T-SQL e PLSQL do professor Felipe Mafra

Aqui você verá 3 bancos de dados diferentes: MySQL, SQL Server e Oracle e também será introduzido às linguagens T-SQL e PLSQL.
Você verá além das diferenças na linguagem SQL, as tarefas de infraestrutura e instalação. 
Como regra de negócios, você aprenderá a diferença entre ambientes OLTP, OLAP e também o que é um Datawarehouse. 
Sim, você será apresentado à sequencia natural de Bancos de Dados Relacionais: Business Intelligence.
